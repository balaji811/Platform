# interface for job  and internship platform 
  Company
  * companies can create a account and post opporiunites.
  * Companies can accept or reject the applications of the students.
  * Companies can chat with the students and can give tasks to test them.

  Student
   * students can create a account or log in
   * students can build resumes and can edit the details
   * students can apply of the jobs posted by the compnay
   * students can also see the recommendations provided by the platfrom
